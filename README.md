#### Installation and Setup
You will need `node` and `npm` installed globally on your machine.

##### Clone:
```bash
git clone https://github.com/singlestone/front-end-technical-exercise-assets.git
cd front-end-technical-exercise-assets
```

##### Installation:

```bash
npm install
```

##### To Start Server:

```bash
npm start
```

##### To Visit App:

`http://localhost:3000`
